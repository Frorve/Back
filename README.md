La aplicación del Backend se ejecutará en: [http://localhost:3000](http://localhost:3000)

## Init
### Ejecutar el contenedor:
```bash
docker compose up --build
```

### Instalación de las dependencias necesarias:
```bash
cd ./api && npm install
```
